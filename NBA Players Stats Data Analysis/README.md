# Exploratory Data Analysis on NBA Players Stats Data

Basic Data Analysis done on NBA Players Stats Data using Pandas
Steps:
  1)Web Scraping from Basket Ball Reference site
  2)Pandas Profiling tool for a quick automatic EDA (installation needed: https://github.com/pandas-profiling/pandas-profiling/archive/master.zip)
  3) Data Cleaning
        a) Finding and Removing Duplicate Rows
        b) Finding Missing Values
        c) Data Imputation using measures of Central Tendency
        d) Dropping unwanted columns
  4) Writing the Web Scraped and cleaned Data into a CSV file and reading it from the CSV (Uploading this CSV Data)
  5) Answering some basic questions with EDA using conditional selection, groupby:
        a) Which player scored the most Points (PTS) Per Game?
        b) Which player scored more than 20 Points (PTS) Per Game?
        c) Which player had the highest 3-Point Field Goals Per Game (3P) ?
        d) Which player had the highest Assists Per Game (AST) ?
        e) Which player scored the highest points (PTS) from the Los Angeles Lakes Team?
        f) Of the 5 positions, which position scores the most?
  6) Finally use basic visualizations like Histogram, Box Plot, Heatmap and Scatter Plot using Pandas inbuilt methods, Seaborn and Matplotlib
